# Saram

## Install
- Install with pip
```
pip3 install -e .
```

## Docs
To set up docs, run the following commands from inside the 
saram directory
```
pip3 install sphinx
cd docs/
make clean html
```

Then open the docs located in `_build/html/index.html`
